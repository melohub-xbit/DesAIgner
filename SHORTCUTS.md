## üí° Pro Tips

### Canvas Management

**1. Quick Navigation**

```
- Use mouse wheel to pan vertically
- Hold Ctrl and scroll to zoom
- Zoom centers on cursor position
```

**2. Precision Editing**

```
- Use Properties Panel for exact values
- Type numbers directly for position/size
- Use arrow keys for 1px movements (future)
```

**3. Layer Organization**

```
- Elements are stacked by creation order
- Later elements appear on top
- Use layer panel to reorder (future)
```

### Collaboration Tips

**1. Real-time Best Practices**

```
- Avoid editing same element simultaneously
- Use different areas of canvas
- Lock elements when not in use
- Auto-save happens every 2 seconds
```

**2. Cursor Following**

```
- See teammates' cursors in real-time
- Colored by user for easy identification
- Username shown next to cursor
```

**3. Project Sharing**

```
- Add collaborators via email
- Set roles: viewer, editor, admin
- All changes sync instantly
```

### Design Workflow

**1. Starting a Design**

```
1. Create project with descriptive name
2. Set canvas background color
3. Enable grid for alignment (coming soon)
4. Start with basic shapes
5. Add text and images
6. Refine with properties panel
```

**2. Efficient Editing**

```
- Select tool (V) for moving/selecting
- Switch tools with keyboard
- Use properties panel for fine-tuning
- Lock elements when done editing
```

**3. Asset Management**

```
- Upload assets before using
- Drag & drop for quick upload
- Thumbnails generated automatically
- Delete unused assets to save space
```

### Performance Tips

**1. Keep Canvas Performant**

```
- Limit elements to ~500 for smooth experience
- Hide complex elements when not editing
- Use simple shapes for placeholders
- Delete unused elements
```

**2. Image Optimization**

```
- Resize images before upload (max 1920px)
- Use web-optimized formats (JPG, PNG, WebP)
- Compress images before upload
- Keep total assets under 50MB per project
```

**3. Collaboration Performance**

```
- Close project when not actively editing
- Limit simultaneous users to 5-10
- Use dedicated editing sessions
- Clear browser cache if sluggish
```

## üé® Design Patterns

### Color Usage

**1. AI Color Suggestions**

```
- Click AI button in toolbar
- Choose base color
- Select scheme (complementary, analogous, triadic)
- Apply to your elements
```

**2. Color Harmony**

```
Complementary: Base + Opposite (high contrast)
Analogous: Base + Adjacent colors (harmonious)
Triadic: Base + Two equidistant colors (balanced)
```

**3. Consistent Palettes**

```
- Pick 3-5 main colors
- Use consistently throughout
- Save favorites for future projects
- Export palette (future feature)
```

### Layout Techniques

**1. Alignment**

```
- Use grid for visual alignment (coming soon)
- Align elements manually
- Use AI auto-align feature
- Group related elements (future)
```

**2. Spacing**

```
- Consistent padding (8px, 16px, 24px)
- Visual breathing room
- Rule of thirds
- Golden ratio for proportions
```

**3. Hierarchy**

```
- Size: Larger = more important
- Color: Contrast for emphasis
- Position: Top/center for focus
- Opacity: Reduce for background
```

## üÜò Quick Reference

### When Something Goes Wrong

**Canvas Issues**

```
Problem: Elements not rendering
Fix: Refresh page, check browser console

Problem: Slow performance
Fix: Reduce elements, hide unused layers

Problem: Can't select element
Fix: Check if locked, try zoom in
```

**Collaboration Issues**

```
Problem: Changes not syncing
Fix: Check internet connection, refresh page

Problem: Can't see collaborator cursor
Fix: Ensure both users in same project

Problem: Lost work
Fix: Check auto-save (2s delay), reload page
```

**Upload Issues**

```
Problem: Image won't upload
Fix: Check file size (<10MB), allowed types

Problem: Thumbnail not generating
Fix: Wait a moment, refresh assets panel
```
