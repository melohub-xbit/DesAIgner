# Team Member Images

Add your team member images to this folder with the following names:

- `team-member-1.jpg` (or `.png`, `.jpeg`, `.webp`)
- `team-member-2.jpg` (or `.png`, `.jpeg`, `.webp`)
- `team-member-3.jpg` (or `.png`, `.jpeg`, `.webp`)

**Recommended specifications:**
- Format: JPG, PNG, JPEG, or WEBP
- Size: Square images work best (e.g., 400x400px or larger)
- Aspect Ratio: 1:1 (square) recommended for best display
- File Size: Optimize images for web (under 500KB recommended)

If images are not found, the About page will display a gradient placeholder with the team member's initial.
